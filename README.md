🇺🇸 Project Introduction (English)

This is a modern Customer Relationship Management (CRM) system built with a separated front-end and back-end architecture, designed for managing customers, business data, and communication within an organization.

🚀 Tech Stack
Backend: ASP.NET Core + Entity Framework Core
Database: SQL Server
Frontend: Vue2 + Axios
Authentication: JWT (JSON Web Token)
Real-time Communication: SignalR (or EChat module)
📌 Features
👤 User authentication & role-based authorization (JWT)
📊 Customer data management (CRUD operations)
📁 Business data tracking
💬 Real-time chat module (EChat)
🔐 Secure API access control
📈 Basic data statistics & analytics
🔄 RESTful API with separated frontend/backend architecture
🏗️ Highlights
Clean and scalable architecture
Token-based secure authentication
Entity Framework Core for database operations
Responsive UI built with Vue2
Extensible real-time messaging module
Suitable for small and medium business CRM systems
